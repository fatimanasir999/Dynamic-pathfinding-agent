# Dynamic-pathfinding-agent
To run the game, first make sure Python (version 3.8 or higher) is installed on your system. Then open Command Prompt or Terminal and install the required library by typing `pip install pygame`. After installing it, download or clone the project and navigate to the folder where the file `Q6_l230714.py` is located using the `cd` command. Run the program by typing `python Q6_l230714.py`. Once executed, a game window will open showing a 25×25 grid where the agent must reach the goal while avoiding obstacles. You can click on grid cells with your mouse to add or remove walls, press **1** to select A* algorithm, **2** to select Greedy Best-First Search, press **H** to switch between Manhattan and Euclidean heuristics, press **R** to generate a random map, and press **SPACE** to start the simulation. The agent will move automatically toward the goal, and if dynamic obstacles appear and block the path, the system will recalculate a new route. The sidebar displays the selected algorithm, heuristic, number of nodes visited, path cost, and execution time as the output of the program.
